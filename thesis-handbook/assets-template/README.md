# 素材模板目录说明

本目录用于作为 `thesis-assets/` 的参考模板，建议在项目开发完成后复制一份并开始填充真实素材。

## 子目录说明

- `screenshots/`
  - 存放页面截图、Network 面板截图、数据库截图等
- `code-snippets/`
  - 存放论文第四章会用到的核心代码片段
- `diagrams/`
  - 存放架构图、流程图、Mermaid 导出的 PNG
- `test-data/`
  - 存放 TTFB、耗时、测试结果截图等

## 推荐素材项

- 简历上传 + 解析结果页
- 流式打字机进行中页面
- 评估报告页
- 用户历史记录页
- Postman 或浏览器 Network 面板的 SSE 截图
- `interview_message` 表数据截图
- SSE 推送、PDF 解析、上下文拼装三类核心代码

## 命名建议

- 截图：`fig4-1-简历解析结果.png`
- 代码片段：`snippet-sse-service.java`
- 测试截图：`ttfb-devtools-2026-04-10.png`
- 文献证据映射表：`literature-evidence-map.md`

## 附加模板

- [dev-notes-template.md](./dev-notes-template.md)：记录真实 Bug、排查过程与修复结论
- [literature-evidence-map-template.md](./literature-evidence-map-template.md)：把文献观点、证据强度、正文落位和引用状态统一记录下来
- [final-evidence-lock-template.md](./final-evidence-lock-template.md)：锁定截图、测试数据、论文和 PPT 是否基于同一版素材
- [figure-table-register-template.md](./figure-table-register-template.md)：登记图号、表号、标题和正文引用状态
- [defense-slide-map-template.md](./defense-slide-map-template.md)：把答辩 PPT 页面与论文、截图、代码和讲解要点对应起来
