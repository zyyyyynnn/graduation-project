# 答辩页映射表模板

本文档用于把答辩 PPT 每一页与论文内容、截图、代码片段、测试数据对应起来，避免答辩展示内容与论文正文、附件材料脱节。

## 使用说明

1. 确定 PPT 大纲后即可填写，不必等 PPT 完全做好。
2. 每一页尽量对应一个核心主题，不要一页塞过多证据。
3. `对应论文章节` 尽量精确到节或小节。
4. `现场讲解要点` 建议控制在 1-3 句，便于答辩时快速回忆。
