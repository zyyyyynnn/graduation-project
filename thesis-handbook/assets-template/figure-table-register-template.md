# 图表编号登记表模板

本文档用于提前登记论文中的图号、表号、图题表题与正文引用状态，避免写完正文后再大规模回填“如图 X-X 所示”“表 X-X 所示”。

## 使用说明

1. 第三章开始画图或整理测试表时就可以填写。
2. 图和表建议分开编号，但可共用一张登记表统一管理。
3. `正文引用状态` 建议使用：
   - 未引用
   - 已引用待核对
   - 已引用并完成核对
4. 若图表改名、换图或调章节位置，务必同步更新本表。
